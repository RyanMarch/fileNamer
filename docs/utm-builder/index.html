UTM Campaign Builder

Discover how to build naming templates that automatically transform inputs into campaign-tagged URLs, and integrate with the file uploader feature.

FileNamer includes dedicated logic to build parameter-tagged URLs instead of standard filenames. If a template’s name contains "UTM" or "URL", the engine shifts into URL Mode.

URL Mode Trigger

Name your custom template with terms like UTM Campaign or Marketing URL Builder to automatically switch the output generator from a filesystem-based path separator to a query string parser.

Configuration Rules

To configure a UTM template, follow these mapping rules in the template editor:

  • Destination URL: Create a standard text input field and include the word URL in its label (e.g. Destination URL). This will serve as the base address of the generated link.
  • Query Parameters: To turn any template field into a URL query parameter, place the query name in parentheses inside the field label. For example:
    • A field labeled Campaign Source (utm_source) becomes ?utm_source=value.
    • A field labeled Ad Type (utm_medium) becomes &utm_medium=value.
  • Spaces and Casing: Enable the lowercase filter to ensure campaign keys and parameters conform to web standard formats. Separation rule spaces will automatically convert to URL-safe separators.

File Uploader Integration

Combining the UTM builder with batch processing provides powerful workflows for marketing and content distribution teams:

1. Renaming Creative Assets to Match Campaign Structures

When uploading creative files (e.g., banner graphics, PDF checklists, newsletter attachments) to an ad manager, CDN, or CMS, it is best practice to keep filenames consistent with campaign attributes.

Use the File Uploader to drop raw file assets, and standard naming filters will convert their local filenames into campaign-indexed names (e.g., newsletter_cpc_spring-launch_blue-button-banner.png) matching your campaign parameters before you batch download.

2. Auto-populating Campaign Fields from Uploaded Filenames

If you upload multiple different creative assets at once, FileNamer can extract their original filenames and inject them into specific query parameters dynamically.

This allows parameters like utm_content to dynamically reference the filename (e.g. utm_content=blue-button-banner), producing a custom tracking URL tailored to each unique creative element.

3. Bulk UTM Generation via CSV URL Lists

If you have dozens of different destination landing pages and need to create campaign tracking links for all of them:

  • Download the CSV template using the action button in the uploader queue.
  • Fill out the destination URLs in the document spreadsheet.
  • Import the file back into the Bulk Edit grid. All UTM queries will be compiled instantly, generating a complete sequence of campaign links in one action.

4. Creating a Link-Asset Registry (Master Sheet)

Once creative assets are named and campaign tracking links are configured inside the Bulk Edit grid, choose the Export Log (CSV) action.

This generates a master registry spreadsheet containing the original filename, the clean campaign-safe target filename (ready for upload to your CDN), and the fully assembled destination UTM link. This tracking document is ready for copy-pasting directly into campaign managers.
