Invoices & Receipts Naming Guide

Simplify tax prep, bookkeeping, and accounting processes with systematically named financial records.

Mishandled receipt names make tracking expenses difficult. The built-in Invoices / Receipts template establishes a structured, searchable format that simplifies search, file retrieval, and expense auditing.

Table of Contents
  • Default Template Structure
  • Best Practices for Customizing

Default Template Structure

The default Invoices / Receipts template uses a hyphen separator (-) and consists of the following components:

  • Date: Date format in YYYY-MM-DD (e.g. 2026-07-09) to enable simple sorting.
  • Vendor: The vendor or provider name (e.g. Google).
  • Invoice Number: The exact invoice or reference number (e.g. INV-98765).
  • Extension: Preserves the original file extension automatically.

Example output filename: 2026-07-09-Google-INV-98765.pdf

Best Practices for Customizing

Personal budgets and corporate finance groups have distinct organization structures. Try adding these custom components:

  • Expense Categories: Add a select/dropdown menu with options like Software, Office, Travel, or Meals to group files visually.
  • Dollar Amount: Add a numeric text input to record the transaction value directly in the filename (e.g., 150USD or 150_00).
  • Tax Status: Add a dropdown for tax years or write-off status (e.g. tax-deductible) to speed up filing preparation.
