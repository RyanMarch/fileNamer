Getting Started Guide

Welcome to FileNamer. Follow this step-by-step tutorial to configure your first file naming template, test options dynamically, and run a safe local batch rename.

FileNamer operates completely inside your web browser. No files are uploaded to any server, ensuring total privacy. By organizing file renaming into visual segments, FileNamer helps establish consistency without complex scripting.

1. Select and Evaluate Presets

When you open the FileNamer application, a default file naming template is pre-selected for you. This allows you to immediately see how segments connect to build a structured name.

  • Explore Existing Presets: Click the Template Picker select dropdown to view built-in presets (e.g., Project Files, Invoices & Receipts, Media Library).
  • Evaluate the Layout: Check the template's structure. If one of the presets fits your general directory needs, you can use it as a starting point.
  • Starting Clean: If none of the presets fit your requirements, click the Add Template button to start with a blank canvas.

2. Customize Fields & Rules

Once a base template is active, use the Field Builder in the left column to add, reorder, and configure individual filename parts.

Structure Control

Reorder parts by dragging them up and down, change text configurations, select output date formats, or add sequential indexes to create unique names.

At the bottom of the Field Builder column, configure formatting rules:

  • Separator: Choose whether to link fields with underscores (_), hyphens (-), dots (.), or no space.
  • Casing style: Enforce consistent lowercase or uppercase transformations across the entire output name.

3. Test Inputs & Preview Outputs

The Preview & Rename panel in the right column adapts dynamically to the fields you define in the builder.

  • Interactive Form: Fill out the input boxes, pick dropdown categories, or select calendar dates.
  • Real-time Preview: The preview box displays the structured filename instantly as you type.
  • Quick Copy: If you only need to rename a single item, type your values and click the button to copy the finished name to your clipboard.

4. Queue Files & Batch Rename

To rename multiple files at once, use the local drag-and-drop workspace:

  • Upload Files: Drop files from your computer into the drop zone area.
  • Compare Names: The table lists your files, displaying the Original Filename on the left and the Renamed Target on the right.
  • Sequential Indexes: If you added an Index segment, adjust the Start Index count box to control the numbering sequence.
  • Bulk Edit Grid: For more detailed mapping, open the Bulk Edit Grid to customize fields for individual files or import values from a CSV template.
  • Download: Click Download Renamed Files to package your structured files into a ZIP archive or download them individually.
