# Filename Maker

A lightweight, local-first web utility for designing standardized naming templates and batch renaming files directly in the browser.

## Features

- **Standardized Templates**: Define rules, select lists, date formats, and separators to enforce naming guidelines.
- **Client-Side File Renaming**: Drag-and-drop or select files to rename them locally. Files never leave your browser.
- **URL-Based Template Sharing**: Serialize naming templates into the URL hash to share with teammates without an account or database.
- **Offline PWA Support**: Install the app and use it completely offline.

## Built With

- Vanilla HTML5 / CSS3 / ES6 Javascript
- esbuild & Wrangler for deployment on Cloudflare Pages
