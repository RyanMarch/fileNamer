Everything happens locally in your browser. FileNamer never uploads your files or filenames to any server — no cloud storage, no third-party processing. Your data stays on your device, which means your naming patterns, file contents, and metadata remain completely private by design. You can even use FileNamer offline once the page has loaded.

Yes. The Templates Hub lets you export any naming pattern as a shareable template file. Your team can import it directly — no account or sync service required. You can lock templates for critical folders to prevent accidental edits, or leave them open for everyone to adapt. The entire library lives in one central hub so naming standards are always a single click away.

FileNamer works with any file type — images, video, audio, documents, archives, code files, and everything in between. Because it runs entirely in the browser, it is platform-agnostic: macOS, Windows, Linux, and ChromeOS are all fully supported without installing anything. The only requirement is a modern browser.
